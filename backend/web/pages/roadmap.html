프로젝트 목표 및 기술 스택

본 프로젝트의 목표는 협업 흐름을 자동화하여 디스코드, GitHub, Notion 등의 다양한 플랫폼 간에 효율적인 정보 공유와 관리 시스템을 제공하는 것입니다.

기술 스택

1. Python 3.12

  • Discord-py-interactions: 디스코드 챗봇 개발용
  • SQLAlchemy: ORM 기반 DB 연동
  • FastAPI: Python 기반 고속 웹 프레임워크
  • Alembic: DB 마이그레이션 관리 도구

2. APIs

  • Discord API: 챗봇 기능 연동
  • Notion API: 페이지 정보 입출력
  • GitHub API: 사용자 및 저장소 정보 조회

3. 기타 도구

  • Nginx: Reverse Proxy 운영
  • Docker: 컨테이너 기반 테스트/배포
  • MySQL: 관계형 데이터 저장소
배포 환경 및 협업 도구

1. 배포 환경

  • GitHub: 코드 배포 및 버전 관리
  • Docker: 일관된 테스트 및 배포 환경
  • Nginx: 프록시 및 트래픽 처리

2. 협업 도구

  • Notion: 문서화 및 일정 관리
  • Discord: 커뮤니케이션 및 실시간 피드백
  • GitHub: 개발 협업 및 코드 공유

이 프로젝트는 오픈소스로 제공되며, 누구나 자유롭게 활용할 수 있도록 설계되었습니다.
