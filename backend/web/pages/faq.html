For other languages, check the GitHub page.

Notion 연동은 어떻게 하나요?

사용자가 직접 Notion Integration을 생성하고 발급된 토큰을 봇에 등록해야 합니다.

How do I invite the bot?

Use the official invite link provided in the README and set a management role with /설정 관리자역할 설정 after inviting.

What commands are available?

Commands like /notion set-token, /notion connect-db and /create-ticket are supported. See the README for a full list.

Where can I ask questions or get support?

Join the Discord support server or DM the bot directly.

Is the project open source?

Yes, visit macqueen0987/notipy on GitHub for the source code.
